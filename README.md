# Bus-Tracking-App
Know where your bus is, before you miss it. Real-time tracking for colleges &amp; schools | Python · JavaScrip
---

## 🎯 The Problem
Students waste time standing at bus stops with no idea when their bus will arrive.
No alerts. No updates. No visibility. Just waiting.

## ✅ Our Solution
A smart bus tracking web app that gives students,
parents, and institutions real-time visibility into their fleet —
from live location to route updates to arrival alerts.

---

## ✨ Key Features
- 📍 **Live Bus Tracking** — Real-time GPS location broadcast via WebSockets
- 🔔 **Proximity Alerts** — Auto-notifies students when bus is within 1km
- 🗺️ **Route Management** — View all routes and stops in one place
- 👨‍💼 **Role-Based Dashboards** — Separate views for Students, Drivers & Teachers
- ✅ **Attendance System** — Auto-marks attendance when student boards
- 🔐 **Secure Auth** — bcrypt password hashing with role-based redirects
- 📱 **Responsive Design** — Works seamlessly on mobile and desktop
- ⚡ **Performance Monitored** — API response alerts if speed exceeds 1.5s

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python, FastAPI |
| Database | SQLite + SQLAlchemy ORM |
| Real-time | WebSockets |
| Authentication | Passlib (bcrypt hashing) |
| Data Validation | Pydantic |
| Location Math | Haversine Formula (GPS distance) |
